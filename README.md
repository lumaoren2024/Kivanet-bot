# Kivanet Bot 1.0 使用教程
``accounts.json``挂机配置，下面是例子！
```json
[
    {
        "Email": "dandan01@gmail.com",
        "Password": "dandan0x1"
    },
    {
        "Email": "dandan02@gmail.com",
        "Password": "dandan0x1"
    }
]
```

``proxy.txt``代理设置，用于挂机使用！
一行一个，例子:
协议://user:pass@ip:port

``reg.txt``自动注册配置
```txt
邮件||||密码||||推荐码||||代理
```
